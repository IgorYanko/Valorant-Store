# Projeto Valorant Store

## Igor Yanko Corrêa Lemes e João Pedro da Costa Figueiredo

### Valorant-API: https://valorant-api.com/

Para rodar o projeto, siga os seguintes passos:
1. Clone o repositório
2. Entre na pasta do projeto
3. Rode **npm install** para instalar as dependências
4. Rode **npm run dev** para iniciar o servidor de desenvolvimento

**Acesso ao projeto:** https://valorant-store-nine.vercel.app/

### Funcionalidades:
- Catálogo de skins com busca e paginação
- Carrossel com skins em destaque
- Página de detalhes com vídeo e variantes (cromas)
- Preços e raridades simulados
